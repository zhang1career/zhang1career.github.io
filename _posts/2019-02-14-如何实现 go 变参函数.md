# 如何实现 go 变参函数

## 1. 实现

直接上码：
```
func demo(params ...interface{}) {
    s := make([]interface{}, 0)
    if len(args) > 0 {
        for _, p := range params {
            s = append(s, p)
        }
    }
    fmt.Println(s)
}
```
